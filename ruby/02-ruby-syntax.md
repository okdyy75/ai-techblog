# Rubyの基本文法とデータ型

## 数値

```ruby
p 1
p 1.5
p 1 + 5
p 1 - 5
p 1 * 5
p 1 / 5
p 1 % 5
```

## 文字列

```ruby
p "こんにちは"
p "こんにちは" + "世界"
```

## 配列

```ruby
p ["Ruby", "PHP", "Python"]
p ["Ruby", "PHP", "Python"][0]
```

## ハッシュ

```ruby
p ({name: "okubo", age: 30})
p ({name: "okubo", age: 30})[:name]
```
